--- layout: devinfo app: ecConfigView --- EC file utility was for development units that have trouble connecting to the DSi Shop. This app can view ec.cfg, delete ec.cfg, shop.log and dev.kp, and delete itself. According to the document EC file utility came with, deleting dev.kp is all that was needed to fix connection issues.

This application was only released to developers with DSi Shop issues, and sharing it among other developers was strictly prohibited. So much so that developers were instructed completely delete the app after using it.
