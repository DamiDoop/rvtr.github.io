Readme for TWL-System Updater Version 1.4a

2010/03/04

Introduction

System Updater is a program for updating the System Menu (DSi Menu) of TWL development kits. It can be used on IS-TWL-DEBUGGER, IS-TWL-CAPTURE, and TWL test units. Because you can select any region while running the program, you can also use System Updater to switch the region setting of your development kits.

Note: System Updater was created with multi-region specifications, so it can be run on the development kits for all regions.

Notes

Please keep the following points in mind when using System Updater.

  • Update to the latest version of the IS-TWL-DEBUGGER software before using this program on the IS-TWL-DEBUGGER hardware.
  • When the System Menu is updated, the unit is automatically initialized. Any game data or save data stored in system NAND memory is erased.
  • When this program is used on a TWL test unit, updating cannot begin if there is insufficient remaining battery life, as indicated by a steady red battery indicator LED. If battery life is extremely low (a blinking red battery indicator LED), then updating cannot begin until the battery has been recharged to a certain extent, even if the AC adapter is connected.
  • Once updated by System Updater, the development kit System Menu (DSi Menu) cannot import or export NAND applications from the System Settings Software Management screen. The System Settings Software Management screen can be used only to delete NAND applications or display their contents.

Procedure for Updating the System Menu (IS-TWL-DEBUGGER)

To update the System Menu of IS-TWL-DEBUGGER, follow this procedure.

  1. Power on the IS-TWL-DEBUGGER hardware and launch the IS-TWL-DEBUGGER software.
  2. Select File → Load File, load SystemUpdater.srl, and execute a hardware reset.
  3. The IS-TWL-DEBUGGER software displays this message: SystemUpdater was loaded. Follow the LCD screen instructions.
  4. Use the IS-TWL-DEBUGGER hardware controller to select the applicable region.
  5. A confirmation screen displays the region you selected. Press the A Button to begin the update. DO NOT turn off the power before the update is completed.
  6. When the update ends successfully, the upper screen displays Update Success!. If the update fails, the upper screen displays Update Failure!. Contact support@noa.com and provide the string displayed on the lower screen.
  7. Power off the IS-TWL-DEBUGGER hardware.
  8. Power on the IS-TWL-DEBUGGER hardware again and configure the DSi Menu initial settings. You cannot proceed with the initial settings if the touch screen's calibration information is invalid. In this case, display the calibration menu by restarting while holding down the L Button, R Button, and START. This allows you to calibrate the system.
  9. Start System Settings from the DSi Menu and confirm the DSi Menu version number shown in the lower-right portion of the upper screen.

Procedure for Updating the System Menu (IS-TWL-CAPTURE and TWL Test Units)

To update the System Menu of IS-TWL-CAPTURE or a TWL test unit, you must write SystemUpdater.srl to a TWL Flash Card.

  1. Power on the IS-TWL-DEBUGGER hardware and launch the IS-TWL-DEBUGGER software.
  2. Select File → Load File and load SystemUpdater.srl. Do not select either of these options: Run TWL-CPUs after loading (X) or Reset TWL-CPUs after loading.
  3. Insert a TWL Flash Card into the IS-TWL-DEBUGGER hardware.
  4. Select Tools → DS Flash Card and write the loaded System Updater program to the TWL Flash Card.
  5. With System Updater written to the TWL Flash Card, insert the card into the IS-TWL-CAPTURE hardware or the TWL test unit. This same method can also be used to update the System Menu of IS-TWL-DEBUGGER hardware.
  6. Subsequent steps are the same as steps 4-8 of Procedure for Updating the System Menu (IS-TWL-DEBUGGER).

Version Number Notation

The version number is displayed in System Settings as follows ("*.*" indicates the version). Because the version number is used for development hardware, "_dev" is shown at the end.

System Updater Optional Features

System Updater has the following optional features.

  • On the confirmation screen displayed after you select the region, you can update the system without installing the two tools TwlNmenu and NandFiler by holding down the SELECT button while you press the A Button to begin the update.
