CTR-SDK 11.6.1 (2022-08-30 - for SystemUpdater 0.35.0) Change Log

2022-08-30

Note that the SystemUpdater for SNAKE that supports CTR-SDK 9.12 and later can only be applied to DP2 development hardware to which SystemUpdater 0.22.32 or later has been applied, or to gamma or later versions of IS-RAY-DEBUGGER.

Changes with a red star (★) in the title are of particular importance, so make sure that you read them.

Changes in CTR-SDK 11.6.0 (2017-09-19 - for SystemUpdater 0.27.0)

CTR-SDK

Additions and Changes

  • 55-S03: (BOSS) Improved documentation for the rules for displaying the update indicator for SpotPass. [SDK]
  • 55-S04: (BOSS) Clarified that the service status of the data store linking task remains as SERVICE_UNKNOWN. [SDK]
  • 55-S06: (EC) Updated the eShop REST API manual. [SDK]
  • 55-S07: (EC) Revised the manual to state that dummy credit cards can no longer be used on South Korean region consoles. [SDK]
  • 55-S08: (FS) Added to the manual that nn::fs::ResultWriteProtected might sometimes not be returned when the SD card is write-protected. [SDK | Sample Demos]

Bug Fixes

  • 55-S01: (APPLET) Fixed a bug that generated an error when jumping to Nintendo eShop while a library applet had been preloaded. [SDK]
  • 55-S02: (BOSS) Fixed a bug that entered an extraneous character at the end of a URL return displayed with BossLotcheckTool. [SDK]
  • 55-S05: (CX) Fixed a bug that caused a buffer overrun with the nn::cx::CompressHuffman function. [SDK]
  • 55-S09: (SND) Fixed a bug where the status for nn::snd::CTR::WaveBuffer structures after waking from sleep mode did not update correctly. [SDK]
  • 55-S10: (TOOL) Fixed a bug that caused insufficient memory with makerom. [SDK]

SystemUpdater

Additions and Changes

  • 55-U01: (OS) Made revisions to make the system more stable. [SystemUpdater]

SampleDemos

Additions and Changes

  • 55-D02: (FS) Added to the manual that nn::fs::ResultWriteProtected might sometimes not be returned when the SD card is write-protected. [SDK | Sample Demos]

Bug Fixes

  • 55-D01: (BOSS) Fixed a bug that could prevent proper handling of task completion when a task error occurred with sample_datastore. [Sample Demos]
